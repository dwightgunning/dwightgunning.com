Just reading 43 Folders and found a link to the GTDTiddlyWiki. It's a Wiki with a built-in framework for using the Getting Things Done productivity system. It really already does what I've been working towards with my JettyJSPWiki. This Wiki is built using just static HTML and JavaScript, requiring no Web Server at all. The best bit is that the download/install involves doing a File > Save As and then loading the local file in your browser. Cool.
