25/02/2005:: Opera Software have released the second beta for the 8th major release of their flagship web browser.

In this Beta, the browser displays security information inside the address bar, located next to the padlock icon that indicates the level of security present on a site....What else to look for in Opera's Beta 2:Easier customization and skinningOnline Certificate Status Protocol (OCSP) verifies that the certificate has not been revoked by the certificate authoritiesAtom newsfeedsBeta 2 is available for download at http://www.opera.com/download/?ver=8.0b2

The user interface has undergone a few subtle improvements after the major overhaul that was seen in version 7. I'm posting this from my 7.54u1 install as Blogger's post editor caused 8.0 bork. So there's obviously some work still to be done. I did noticed that the javascript toolbar in the editor had a few more HTML tag insertion options for bold, italic, etc which I don't get on 7.54 under linux or windows, so perhaps they're implementing some more IE extensions.On the topic of using an alternative browsers; I'm dissapointed with having to use IE for all intranet sites at work. There's heaps of ASP and wacky javascript making it impossible to use anything but IE. Not only do the sites render incorrectly (I could deal with that alone), but many of the links and mouse-over menu's are totally broken.
